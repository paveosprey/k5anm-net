**Programming Guides by Model**

Properly programmed radios are essential for emergency communications — a radio with the wrong frequencies, tones, or settings is as useless as no radio at all. This section provides step-by-step programming guides for radios commonly used in Travis County ARES and RACES operations, including CHIRP codeplug templates, repeater frequencies, CTCSS/DCS tone assignments, and model-specific tips. Guides are written for operators of all experience levels and updated to reflect current Travis County repeater infrastructure.
