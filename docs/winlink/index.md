**Winlink Digital Email over Radio**

Winlink is a global radio email system that allows amateur operators to send and receive email messages without internet infrastructure — making it a critical tool for emergency communications when conventional networks fail. Used extensively by ARES and RACES operators during activations, Winlink enables structured message passing to served agencies using standard ICS forms. This section covers Winlink Express setup, VARA HF and FM modem configuration, gateway selection, message templates, and operational use within Travis County emergency communications exercises and activations.
